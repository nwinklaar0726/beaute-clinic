# Video del Hero

## Instrucciones

Para agregar el video de fondo al Hero (estilo Bellas Place):

1. **Coloca tu video** en esta carpeta: `public/videos/`
2. **Nombre del archivo**: `beaute-hero.mp4`
3. **Formato recomendado**: MP4 (H.264 codec)
4. **Resolución**: 1920x1080 (Full HD) o 3840x2160 (4K)
5. **Duración**: 10-30 segundos (loop)
6. **Tamaño**: Optimizado para web (< 10MB ideal)

## Fallback

Si no hay video disponible, se muestra automáticamente la imagen:
`/images/hero-spa.jpg`

## Controles

El Hero incluye controles para:
- Play/Pause
- Mute/Unmute

## Recursos de video gratuitos

Si necesitas videos de stock:
- Pexels: https://www.pexels.com/search/videos/spa/
- Pixabay: https://pixabay.com/videos/search/spa/
- Coverr: https://coverr.co/s?q=spa

Busca videos de:
- Spa treatments
- Wellness center
- Luxury beauty
- Massage therapy
- Relaxation

## Tips

1. Usa videos con movimiento suave y lento
2. Evita cambios de escena bruscos
3. Colores cálidos y relajantes funcionan mejor
4. El video debe ser loop-friendly (sin cortes visibles)
